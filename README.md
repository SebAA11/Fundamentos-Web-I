# Fundamentos-Web-I

Taller Inicial 

1. ¿Qué es HTML y cual es su función en la web?
HTML es el lenguaje básico que se utiliza para crear la estructura de las páginas web. Su función principal es organizar el contenido, como textos, imágenes, enlaces y formularios, de manera que el navegador pueda interpretarlo y mostrarlo correctamente al usuario.
2. ¿Qué es una etiqueta HTML? Menciona las etiquetas mas comunes.
Una etiqueta HTML es una instrucción que indica al navegador cómo debe mostrarse un contenido. Las más comunes son <html>, <head>, <body>, <h1> hasta <h6>, <p>, <a>, <img>, <div> y <span>, ya que permiten organizar textos, imágenes y enlaces dentro de la página.
3. ¿Que es un atributo de una etiqueta HTML? Menciona los atributos mas comunes
Un atributo es una característica adicional que se le da a una etiqueta para modificar su comportamiento o apariencia. Algunos de los más usados son id, class, src, href, alt y style, los cuales ayudan a identificar elementos, agregar estilos o enlazar recursos.
4. ¿Qué es CSS y como se utiliza para el diseño web?
CSS es el lenguaje que permite darle estilo visual a una página web. Se utiliza para definir colores, tamaños, posiciones y estilos de los elementos, haciendo que la interfaz sea más atractiva, ordenada y agradable para el usuario.
5. ¿Qué es una propiedad en CSS? Menciona las propiedades mas comunes.
Una propiedad en CSS define una característica visual de un elemento. Algunas muy usadas son color, background-color, font-size, margin, padding, border, width y height, las cuales permiten ajustar la apariencia y el diseño general.
6. ¿Qué es un selector en CSS y cuales tipos existen?
Un selector permite indicar qué elementos del documento se van a modificar. Existen selectores por etiqueta, por clase, por identificador y selectores más específicos que permiten aplicar estilos de forma precisa según la necesidad del diseño.
7. ¿Qué es JavaScript y como añade interactividad a las paginas web?
JavaScript es un lenguaje que permite crear comportamientos dinámicos en una página web. Gracias a él, es posible validar formularios, mostrar mensajes, crear animaciones, responder a eventos del usuario y actualizar información sin recargar la página.
8. ¿Cuáles son los tipos de datos primitivos en JavaScript?
Los datos primitivos son los valores básicos que maneja el lenguaje, como números, texto, valores lógicos (verdadero o falso), valores indefinidos y valores nulos, los cuales permiten representar información simple dentro de los programas.
9. ¿Cómo funcionan las estructuras de control como if, else, switch y bucles en JavaScript?
Estas estructuras permiten controlar el flujo del programa, es decir, decidir qué acciones se ejecutan según ciertas condiciones o repetir instrucciones varias veces, logrando que los programas sean más flexibles y eficientes.
10. ¿Por qué es importante usar nombres significativos para variables y metodos?
Porque facilitan la comprensión del código, tanto para el propio desarrollador como para otras personas. Nombres claros ayudan a identificar rápidamente la función de cada elemento y reducen errores durante el mantenimiento.
11. ¿Qué es una variable de entorno y porque es importante en JavaScript o la programación en general?
Una variable de entorno permite almacenar configuraciones externas al código, como claves, rutas o configuraciones del sistema. Esto mejora la seguridad, la organización y la portabilidad de los proyectos.
12. ¿Qué son las herramientas de desarrollo de Chrome y como se accede a ellas?
Son un conjunto de utilidades que ofrece el navegador para analizar, depurar y optimizar páginas web. Se accede presionando F12 o con clic derecho y seleccionando “Inspeccionar”.
13. ¿Qué se puede hacer en el panel “Elements” de las herramientas de desarrollo?
Permite visualizar y modificar en tiempo real la estructura y los estilos de una página web, lo cual facilita la corrección de errores y la mejora del diseño.
14. ¿Cómo se utiliza el panel “Console” de las herramientas de desarrollo y para que sirve?
Se usa para mostrar mensajes, detectar errores y probar instrucciones de JavaScript, lo que ayuda a verificar el correcto funcionamiento del código.
15. ¿Qué información se puede obtener del panel “Network” y por que es importante?
Permite observar las solicitudes que hace la página al servidor, como la carga de imágenes, archivos y datos. Es importante para analizar tiempos de carga, detectar errores de conexión y mejorar el rendimiento.
